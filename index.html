Que puedo hacer?

HTML

Puedo formatear documentos con encabezados, párrafos, listas y otros elementos, así mismo, también puedo incrustar imágenes, videos, archivos de audio y otros elementos multimedia a través de hipervínculos y vincular otras páginas web en el mismo sitio web o desde otro.

CSS

Puedo definir y crear la presentación visual de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML, creando experiencias web dinámicas y atractivas.

JAVASCRIPT

Puedo modificar dinámicamente HTML y CSS para actualizar una interfaz de usuario, a través de una API del modelo de objetos del documento.
